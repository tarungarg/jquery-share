jQuery share plugin demo

Share is very popular plugin for sharing the link at facebook, twitter, tumblr etc.
Now you can also share on pintrest, It will no more give you the error of image url. Here is the example give below


Steps to share
  • Include the jquery share css and js into the file.
  • Take a div with attribute "id" in the body.
  • Call share method on that id.
  • Set networks on which you want to share your link.
  • Now set "urlToShare" attribute.
  • One extra attribute added is media. This attribute is for pint it the image on pintrest.
    $("#social-post").share({
    networks: ['facebook', 'twitter', 'pinterest', 'tumblr'],
    urlToShare: "https://github.com/iatek/jquery-share",
    media: "http://placehold.it/350x150&text=FooBar1" // image path for pintrest to pin
    });
